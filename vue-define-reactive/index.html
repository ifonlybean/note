问题一：vue为什么无法响应数组属性和length的变化

实战考察

总结

  1. Vue不能检测以下数组值以及length的变化，这并不全是由于Object.defineProperties方法导致的，解决方案可以使用vm.$set或者push、pop、shift、unshift、splice、sort、reverse这7种方法解决；

  2. 如果数组的每项满足isObject的判断条件，也是可以响应式的；

  3. 直接对数组进行重新赋值也可以实现响应式；

  4. 最后，还可以使用vue提供的$set方法；
