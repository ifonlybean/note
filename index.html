使用Hexo主题搭建个人github博客

Posted on 2020-04-29 | Post modified: 2020-04-29

说明

首先在 github 上新建两个代码仓库，例如 blog 和 blog-origin 两个代码库，其中 blog 代码库是最终站点的代码，blog-origin 是基于hexo主题的源代码（当然也可以只建一个代码库，不过会比较混乱，在提交时也会有问题，因此还是建议分开两个代码库）

说明：github 代码仓库的安装假装大家都会了，如有需要再补充~

安装

1、blog-origin代码库

补充说明：

  • 每次发布代码需要执行hexo g 和 hexo d 两个命令，比较麻烦，可以修改 package.json 文件增加自定义指令，例如修改 “deploy”: “hexo g && hexo d”，这样修改完代码发布的时候只需要执行 npm run deploy 即可，可根据个人习惯自行修改设置；

  • 如果推送完代码到github上没有生效，请强制刷新页面；

2、blog代码库

  • 代码：blog的代码是由blog-origin编译得来，通过hexo d的方式推送到github代码库中，不用手动处理~

  • 设置：需要在 Settings 里 GitHub Pages里选择对应的分支就会得到默认的站点地址~

更换Hexo主题
